gulp-starter (v1.5.3)

A file processing boilerplate for HTML, Sass, JS, PNG, JPEG, GIF, and SVG files powered by Gulp.js and a collection of useful plugins.

Get started quickly with this template. For more information, visit GitHub.

Quickstart Guide

  1. Clone or download this Git repo onto your computer.
  2. Install Node.js if you haven't already.
  3. Run npm install to install the required dependencies.
  4. Run gulp to start the default Gulp task.

Gulp Plugins

This project is configured with the following Gulp plugins to automate tasks:

  • browsersync - Local server with live-reloading.
  • gulp-sass - SCSS to CSS compilation.
  • autoprefixer - Automatically adds vendor prefixes to CSS.
  • gulp-rename - Renames compressed CSS/JS files.
  • gulp-cssnano - Minifies the CSS file.
  • gulp-babel - Transpiles JavaScript with Babel.
  • gulp-concat - Concatenates JavaScript files.
  • gulp-uglify - Minifies JavaScript files.
  • gulp-imagemin - Compresses images (JPG, PNG, GIF).
  • gulp-cache-bust - Cache busting for resources.
  • Final files are moved to the /dist folder for easy deployment.
